SIMPLE-SCADA


Simple-Scada – предназначена для разработки и обеспечения работы в реальном времени систем сбора, обработки, отображения и архивирования информации об объекте мониторинга или управления. Главная цель проекта – это простота и удобство для конечного пользователя.


Системные требования:


*Операционная система — Windows XP/7/8/10 x86 или x64, Windows Embedded x86/x64;

*Процессор — Intel Atom D2550 и выше;

*Видеокарта — Intel GMA 3650;

*1 Гб ОЗУ;

*Звуковая карта;

*DirectX 9.0c или выше / OpenGL;

*Пакет «.NET Framework 4.5» — необходим для работы системы отчетов.

ВНИМАНИЕ! На Windows XP возможна установка СУБД MySQL версии не выше 5.5, в то время как для работы с Simple-Scada требуется версия MySQL не ниже 5.6.2. Поэтому, для работы в качестве сервера с архивированием данных ПК с Windows XP + СУБД MySQL не подойдет.
